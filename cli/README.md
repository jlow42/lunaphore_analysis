# SPARC CLI

Command-line entry points for running SPARC pipelines and utilities.

Planned commands include:
- `sparc run` – execute an end-to-end analysis using a YAML config.
- `sparc ingest` – convert OME-TIFF to OME-Zarr and register metadata.
- `sparc status` – query job states and history.

Implementation guidance is documented in `../docs/system_overview.md`.
